# Electronics Store

## Overview

Electronics Store is a .NET C# application developed in Visual Studio for managing electronic products and store inventory. The system provides an easy way to add, update, delete, and view product information.

## Features

* Add new electronic products
* View all products
* Update product details
* Delete products
* Search for products
* Manage inventory records

## Technologies Used

* C#
* .NET
* Visual Studio
* SQL Server 

## Requirements

* Visual Studio 2022 or later
* .NET SDK
* SQL Server
